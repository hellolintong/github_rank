得分计算公式

用户得分 = 跟随用户的数量 * 0.４ + （原创项目的得分 + fork项目数量） * 0.3 + 组织的得分 * 0.3
组织得分 = 项目数量 * 0.7 + 成员数量 * 0.3
项目得分 = star数量 + fork数量 + 1
